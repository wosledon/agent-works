# Task-M3-009: 字段映射配置组件

**优先级**: P0
**预估工时**: 4h
**依赖**: Task-M3-008
**认领人**: 待认领
**状态**: 待开始

## 描述
前端任务。实现图表字段映射配置组件，根据图表类型（柱状图、折线图、饼图等）动态展示可配置的映射项。

## 验收标准
- [ ] 根据图表类型动态生成映射配置项（X轴、Y轴、系列、数值等）
- [ ] 映射字段下拉选择（从查询结果列中选择）
- [ ] 支持多系列配置（Y轴可添加多个）
- [ ] 支持数据格式配置（数值格式化、日期格式）
- [ ] 实时预览绑定效果
- [ ] 映射配置自动保存
