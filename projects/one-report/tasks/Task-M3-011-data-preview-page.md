### 更新文件

### 页面功能
1. **顶部工具栏**
   - 数据源下拉选择（显示所有已配置数据源，带类型标签）
   - 执行查询按钮（支持 Ctrl+Enter 快捷键）
   - 刷新按钮
   - 清空按钮
   - 全屏切换按钮

2. **参数区**
   - 自动从 SQL 中提取 `${paramName}` 格式的参数
   - 支持设置参数类型（文本/数字/日期）
   - 参数输入验证
   - 独立的执行按钮

3. **数据表格区**
   - 使用 DataPreviewTable 组件（虚拟滚动、排序、列宽调整）
   - 加载状态显示

4. **统计信息**
   - 总行数显示
   - 执行时间（毫秒）
   - 当前数据源名称

5. **错误处理**
   - 错误提示区域
   - 可关闭的错误消息
   - 错误详情显示

6. **用户体验**
   - 切换数据源时自动清空数据
   - 加载状态防止重复提交
   - 返回按钮支持
   - 响应式布局（参数区自适应宽度）
   - Fluent Design 风格一致
