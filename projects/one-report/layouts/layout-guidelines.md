## 整体布局架构

### 应用框架

```
┌─────────────────────────────────────────────────────────────┐
│  App Header (48px)                                          │
│  ├─ Logo + 产品名称                    ├─ 全局操作区        │
│  └─ 主导航（设计器 | 数据源 | 导出）   │  预览 | 保存 | 设置 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Main Content Area                                          │
│  (根据路由显示不同页面)                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 全局顶部栏

**高度**: 48px
**背景**: `--color-neutral-0`
**底部边框**: 1px solid `--color-neutral-20`

#### 左侧区域
- **Logo**: 32px × 32px，距左 16px
- **产品名称**: "One-Report"，16px 半粗体
- **主导航**: 距产品名称 48px
  - 选项：设计器 | 数据源 | 导出
  - 选中状态：品牌色下划线 2px
  - 间距：选项之间 32px

#### 右侧区域
- **全局操作**: 距右 16px
- **按钮顺序**: 预览（次要）| 保存（主要）
- **设置图标**: 最右侧，20px

### 2. 中央画布区域 (自适应)

#### 画布背景
- 颜色: `#FAFAFA`
- 网格: 20px 点阵
- 网格颜色: `#E8E8E8`
- 页面阴影: `--shadow-16`

#### 页面规格

**A4 纵向**
- 尺寸: 794px × 1123px (96 DPI)
- 边距: 默认 40px
- 背景: 白色

**A4 横向**
- 尺寸: 1123px × 794px

**Letter 纵向**
- 尺寸: 816px × 1056px

#### 画布工具栏

位置: 画布右下角，距边 16px

```
┌──────────────────────────────────────┐
│ -  ████  +  |  100%  |  ⊡ 适应  |  ☐ 实际 │
└──────────────────────────────────────┘
```

**控件说明**
| 控件 | 功能 |
|------|------|
| `-` | 缩小 (50% - 200%) |
| `+` | 放大 |
| `100%` | 当前缩放比例显示 |
| `⊡ 适应` | 适应屏幕宽度 |
| `☐ 实际` | 实际大小 (100%) |

#### 组件选中状态

**选中框样式**
- 边框: 2px solid `--color-brand-100`
- 调整手柄: 8px × 8px 正方形
- 手柄位置: 四角 + 四边中点
- 手柄颜色: 白色填充，品牌色边框
- 旋转手柄: 顶部中心，20px 高度

**多选状态**
- 整体选择框包围所有选中组件
- 各组件显示虚线边框

**拖拽预览**
- 半透明复制: `rgba(15, 108, 189, 0.15)`
- 显示对齐辅助线

#### 对齐辅助线

**智能对齐**
- 与邻近组件边缘对齐时显示
- 与画布中心对齐时显示
- 等间距提示

**辅助线样式**
- 颜色: `--color-brand-100`
- 宽度: 1px
- 虚线: 2px dash / 2px gap

## 响应式适配

### 断点定义

| 断点 | 宽度 | 说明 |
|------|------|------|
| Desktop XL | ≥ 1920px | 大屏工作站 |
| Desktop | 1440px - 1919px | 标准桌面 |
| Desktop S | 1280px - 1439px | 小屏桌面 |
| Tablet | 1024px - 1279px | 平板/小笔记本 |
| Mobile | < 1024px | 不支持完整功能 |

### 适配策略

#### Desktop XL (≥ 1920px)
- 画布默认显示 100%（无需缩放）
- 属性面板可展开更多选项
- 画布工具栏可显示更多快捷操作

#### Desktop S / Tablet (1024px - 1439px)

**侧边栏折叠**
```
┌────────┬────────────────────────────────┬──────────┐
│  ≡     │                                │          │
│        │         Canvas                 │  Props   │
│  (32px)│                                │ (280px)  │
│        │                                │          │
└────────┴────────────────────────────────┴──────────┘
```
- 左侧组件面板折叠为图标栏 (32px)
- 点击图标展开面板（覆盖模式）
- 属性面板保持展开，宽度压缩至 280px

**画布缩放**
- 默认适应屏幕宽度
- 最小缩放 50%

#### Mobile (< 1024px)

**不支持完整编辑功能**
- 显示简化预览界面
- 提示"请在桌面端编辑"
- 仅支持查看和导出历史报表

### 最小推荐尺寸

- **最佳体验**: 1440px × 900px
- **最低支持**: 1280px × 720px
- **低于最低**: 显示尺寸警告提示
