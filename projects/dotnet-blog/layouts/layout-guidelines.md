## 1. 布局原则

### 1.1 栅格系统

采用 12 列栅格系统，配合最大宽度约束：

```
容器宽度: max-width 1200px
列间距 (gap): 24px
行间距 (row-gap): 32px
侧边栏宽度: 280px (固定)
```

### 1.2 内容区域

| 布局类型 | 内容区宽度 | 侧边栏 | 适用页面 |
|---------|-----------|--------|---------|
| 全宽 | 100% | 无 | Hero 区域、页脚 |
| 居中 | max 720px | 无 | 文章详情、纯内容页 |
| 左内容右栏 | flex: 1 | 280px | 首页、文章列表 |
| 左栏右内容 | 240px | flex: 1 | 后台管理 |

### 1.3 页面结构模板

```
┌─────────────────────────────────────────────────────────┐
│                    导航栏 (64px)                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                     Hero 区域                            │
│                   (可选, 首页专用)                        │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    主内容区域                            │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                      页脚区域                            │
└─────────────────────────────────────────────────────────┘
```

### 2.2 Hero 区域规范

**尺寸:**
- 高度: `320px` (桌面) / `240px` (移动端)
- 背景: 渐变或图片 + 深色遮罩
- 内容垂直水平居中

**元素:**
| 元素 | 样式 | 尺寸/间距 |
|------|------|----------|
| 博客标题 | 字重 700, 白色 | 48px / 32px(移动) |
| 副标题 | 字重 400, 白色 80% | 20px |
| CTA 按钮 | Primary 按钮 | margin-top: 32px |

**Fluent 效果:**
```css
.hero {
  background: linear-gradient(135deg, #0078D4 0%, #106EBE 100%);
  position: relative;
}
.hero::before {
  content: '';
  position: absolute;
  inset: 0;
  background: url('/noise.png') opacity(0.03);
}
```

## 6. 响应式适配

### 6.1 断点策略

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| Desktop | >= 1280px | 完整布局 |
| Laptop | 1024px - 1279px | 侧边栏宽度缩小 |
| Tablet | 768px - 1023px | 隐藏侧边栏，汉堡菜单 |
| Mobile | < 768px | 单列布局，底部导航 |

### 6.2 移动端适配

**导航栏:**
- 高度: `56px`
- Logo + 汉堡菜单 + 操作按钮

**侧边栏:**
- 改为从左侧滑出的 Drawer
- 宽度: `280px`
- 遮罩层: `rgba(0,0,0,0.5)`

**文章列表:**
- 卡片单列显示
- 缩略图全宽在上

**文章详情:**
- 标题字号: `24px`
- 正文: `16px`
- 代码块: 横向滚动

**后台管理:**
- 侧边栏收起为图标栏 (64px)
- 表格改为卡片列表
- 表单项单列堆叠
